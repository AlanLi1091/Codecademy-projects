HTML Table Cheat Sheet

Here are some useful HTML tags for tables.

Tag Function
<table> Creates a table
<tr> Adds rows
<td> Adds data to a row
<th> Table heading
colspan The span of data column
rowspan The span of data row
Three main sectors
  • <thead>: Table's head
  • <tbody>: Table's body
  • <tfooter>: Table's footer
